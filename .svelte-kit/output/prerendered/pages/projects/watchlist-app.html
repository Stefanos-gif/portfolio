Watchlist App

This watchlist app lets you create unlimited lists for movies, TV shows, books, travel spots—anything you want to track.
View the source code

Pilot Testing: The app is currently in pilot testing, and we are expecting 50 users by the end of the year.

Description

A secure, fast watchlist manager with Google sign-in, Supabase backend, and Vercel deploy. Organize recommendations, track progress, and sync across devices.

Technologies Used

  • SvelteKit + TypeScript
  • Supabase (PostgreSQL)
  • Google OAuth for authentication
  • Vercel for hosting

The Problem

People collect recommendations everywhere—notes, chats, tabs—and then lose them. They need one simple place to store, organize, and complete items.

My Solution

A unified watchlist app with unlimited lists, quick add, progress tracking, notes/links, and cross-device sync.

Core Features

  • Unlimited lists: Movies, shows, books, restaurants, travel, learning goals, shopping—anything.
  • Fast item management: Add title, notes, and an optional URL; mark complete when done.
  • Stay organized: Rename, delete items, collapse lists, and keep completed items for reference.
  • Sync everywhere: Your data is available across phone, tablet, and desktop.

Security & Privacy

  • Google OAuth sign-in (no extra passwords)
  • HTTP-only cookies for session protection
  • Data stored on Supabase (managed Postgres) with backups
  • Encrypted transport (HTTPS) and Vercel’s global infrastructure
  • Private by default; no tracking or data selling

Architecture

  • Frontend: SvelteKit + TypeScript
  • Auth: Google OAuth → secure, HTTP-only session cookie
  • Backend/Data: Supabase PostgreSQL
  • Hosting: Vercel
